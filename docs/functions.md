# アプリケーション機能一覧

## YouTube動画検索機能
- **FR-010:** 自然言語による動画検索
- **FR-020:** 検索結果の表示（タイトル、サムネイル、公開元など）
- **FR-030:** 選択した動画の再生準備

## 動画再生およびメモ機能
- **FR-040:** アプリ内での動画再生
- **FR-050:** Markdown形式でのメモ入力・編集
- **FR-060:** 整形されたメモの表示
- **FR-070:** メモのローカルファイルへの保存
- **FR-080:** 保存場所の選択
- **FR-090:** (将来的な拡張) 保存済みメモの読み込み

## AIによる動画要約機能
- **FR-100:** ワンタッチでの要約生成
- **FR-110:** 生成された要約の表示
- **FR-120:** トランスクリプトを利用した要約
